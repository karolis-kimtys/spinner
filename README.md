# Simple Customizable React Spinner Component

## Install using:

### `npm install spinner`
or
### `yarn add spinner`


### `import Spinner from 'Spinner';`

### 
`<Spinner
    loaderTitle="Loading....."
    fontSize="50px"
    position="center"
    fontColor="blue"
    spinnerWidth="50px"
    spinnerHeight="50px"
    fontFamily="inherit"
      />`      

**Enjoy!**
