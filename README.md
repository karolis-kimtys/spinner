# Simple Customizable React Spinner Component

## Install using:

### `npm install mtm-spinner`
or
### `yarn add mtm-spinner`


```
import Spinner from 'mtm-spinner';
```


```
<Spinner
    loaderTitle="Loading....."
    fontSize="50px"
    position="center"
    fontColor="blue"
    spinnerWidth="50px"
    spinnerHeight="50px"
    fontFamily="inherit"
    />
```  

**Enjoy!**
